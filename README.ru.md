## Введение
Я захотел попробовать воссоздать легендарный DOOM, используя С (стандарт C23) и
raylib.  
Вся графика рисуется на процессоре, потому что я также хотел это попробовать.

Весь код написан в среде разработки CLion (JetBrains). Сборка через CMake на
WSL2. В директории [`.idea/`](.idea) лежат настройки среды разработки.
